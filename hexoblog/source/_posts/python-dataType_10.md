<h3>list, dict, set의 차이</h3>
1. 사용가능한 메서드가 차이난다.<br>
2. 타입이 다르다.<br>
3. dict의 경우 키, 밸류가 존재한다.<br>

<h3>사용가능한 메서드</h3>
dict : keys, values, get, update, pop, clear ..<br>
list : append, index, pop, extend, insert ..<br>
set : 집합개념 union, add, intersection, dieffernce ..
